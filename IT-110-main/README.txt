Project: LYRIN Translator (Single-Page Application)

Main Files:
- index.html (Portfolio homepage with prominent link to the app)
- app.html (Main application SPA)
- styles.css (Styles)
- script.js (JavaScript behavior and API integration)
- translator.js (Translation helper functions and API wrappers)

External API Used:
- Name: MyMemory Translation API
- Docs: https://mymemory.translated.net/doc/spec.php

Key Features:
- Clean, responsive UI (Flexbox/Grid)
- Input fields for text and language selection
- Realtime translation with async/await and fetch()
- Dynamic DOM updates for results
- Error handling with user-friendly messages

How to Run:
1) Open index.html to view the homepage.
2) Click "Open the App" to launch app.html.
3) Type text, choose languages, and view translation.

Notes:
- Internet connection required for online translation.
- Fallback/demo test pages: test.html, test-mymemory.html, test-fallback.html (optional).
