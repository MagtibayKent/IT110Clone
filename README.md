## Features Overview

### Sidebar
- Dark background (#131112)
- Vertical icon layout with hover effects
- Icons: Translate, Help, Trophy, Book
- Interactive click handlers

### Main Interface
- Clean LYRIN header with modern typography
- Two-panel translation interface
- Language dropdown selectors (8 languages)
- Swap button for language switching
- Real-time translation with LibreTranslate API

### Responsive Design
- Mobile-first approach
- Flexible layout for all screen sizes
- Touch-friendly interface
- Optimized for tablets and phones

### API Integration
- **LibreTranslate API** - Real translation service
- **Error handling** - Graceful fallbacks
- **Loading states** - Visual feedback
- **Debounced input** - Efficient API calls

## Usage

1. **Open `index.html`** in any modern browser
2. **Type in the left panel** - Translation appears automatically
3. **Use the swap button** - Swaps languages and content
4. **Change languages** - Use the dropdown selectors
5. **Test responsiveness** - Resize your browser window
